# دليل التشغيل — نسخة Gemini المجانية

## الملفات
```
fundamental_bot_gemini/
├── main.py           ← الكود الرئيسي
├── requirements.txt  ← المكتبات
├── railway.toml      ← إعدادات Railway
└── system_prompt.txt ← البرومبت (انسخه من ملف system-prompt.txt)
```

---

## الخطوة 1 — Gemini API (مجاني)
1. اذهب إلى: aistudio.google.com
2. سجل بحساب Google
3. اضغط: Get API Key
4. احفظ المفتاح

---

## الخطوة 2 — Telegram Bot (مجاني)
1. افتح Telegram → ابحث: @BotFather
2. أرسل: /newbot
3. اختر اسم للبوت
4. احفظ التوكن

5. لمعرفة Chat ID:
   - ابحث: @userinfobot
   - أرسل أي رسالة
   - احفظ الرقم

---

## الخطوة 3 — NewsAPI (مجاني)
1. اذهب إلى: newsapi.org
2. اضغط: Get API Key
3. سجل بإيميلك
4. احفظ المفتاح

---

## الخطوة 4 — نسخ البرومبت
1. افتح ملف system-prompt.txt
2. انسخ كل المحتوى
3. أنشئ ملف جديد: system_prompt.txt
4. ضعه في نفس مجلد main.py

---

## الخطوة 5 — رفع على GitHub
1. اذهب إلى: github.com
2. أنشئ حساب مجاني
3. اضغط: New repository
4. اسمه: fundamental-bot
5. ارفع الملفات الخمسة

---

## الخطوة 6 — Railway
1. اذهب إلى: railway.app
2. سجل بحساب GitHub
3. اضغط: New Project
4. اختر: Deploy from GitHub repo
5. اختر: fundamental-bot

---

## الخطوة 7 — المتغيرات السرية في Railway
اذهب إلى Variables وأضف:

```
GEMINI_API_KEY    = (مفتاح Google AI Studio)
TELEGRAM_TOKEN    = (توكن BotFather)
TELEGRAM_CHAT_ID  = (رقمك من userinfobot)
NEWSAPI_KEY       = (مفتاح NewsAPI)
```

---

## الخطوة 8 — التشغيل
1. اضغط: Deploy
2. انتظر دقيقة
3. ستصلك على Telegram:
   "✅ نظام التحليل الأساسي بدأ العمل"
